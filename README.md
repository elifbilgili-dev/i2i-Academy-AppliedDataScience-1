# Applied Data Science - Classification Comparison

This project was developed as part of the i2i Academy Applied Data Science homework.

## Objective

The goal of this project is to compare the performance of two different machine learning classification algorithms on the same dataset.

## Dataset

Breast Cancer Wisconsin Dataset

The dataset is used to classify breast tumors as benign or malignant.

## Machine Learning Pipeline

- Load the dataset
- Clean missing values
- Feature selection
- Train/Test split
- Feature scaling using StandardScaler
- Train two classification models
- Evaluate the models

## Algorithms

- Logistic Regression
- K-Nearest Neighbors (KNN)

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix

## Result

| Model | Accuracy |
|--------|----------|
| Logistic Regression | 96.43% |
| K-Nearest Neighbors (KNN) | 97.14% |

K-Nearest Neighbors (KNN) achieved the best performance on this dataset.
